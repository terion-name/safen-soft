Защита специализированного ПО

Новости 14 / 06 / 2016

В SafenSoft TPSecure возможна гибкая настройка пользовательских правил как для отдельных приложений, так и для их групп. Это позволяет SafenSoft TPSecure интегрироваться со специализированным ПО (например, с АСР банкомата) для контроля его целостности, защиты от изменения программного кода, защиты хранимых локально данных, включая временные файлы приложений.
